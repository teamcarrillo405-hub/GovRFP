Role Access Level Description
Admin
Full Access
Complete control over all settings, members, billing, integrations, proposals, and analytics. Can manage roles and permissions.
Capture Manager
Pipeline + Proposals
Full access to pipeline and proposal management. Can create, edit, and submit proposals. Read access to analytics and library.
Proposal Writer
Proposals Only
Can create and edit proposal sections assigned to them. Access to content library for reference. Cannot submit or delete proposals.
Compliance Reviewer
Read + Comment
Read-only access to all proposals with ability to leave compliance comments and flag sections. Cannot edit or create proposals.
Pricing Analyst
Analytics + Pricing
Access to pricing modules, analytics dashboard, and cost volume sections. Can generate pricing reports and review competitive analysis.
