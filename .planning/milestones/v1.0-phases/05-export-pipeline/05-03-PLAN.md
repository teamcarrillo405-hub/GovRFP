Exports:
1. `tiptapToPdfElements(sectionName: string, content: JSONContent | null): React.ReactElement[]` — walks JSONContent recursively:
   - `'doc'`: flatMap children
   - `'heading'`: `React.createElement(Text, { style: styles.h1|h2|h3 }, ...textChildren)`
   - `'paragraph'`: `React.createElement(Text, { style: styles.paragraph }, ...textChildren)`
   - `'bulletList'`: map listItem children, prefix each with bullet char `"\u2022 "`
   - `'orderedList'`: map listItem children, prefix each with `"${index + 1}. "`
   - `'table'`: `React.createElement(View, null, ...rows)` where each row is `React.createElement(View, { style: styles.tableRow }, ...cells)` and each cell is `React.createElement(View, { style: styles.tableCell }, ...cellContent)`
   - `'text'`: `React.createElement(Text, { style: computedStyle }, node.text)` where computedStyle applies `fontWeight:'bold'`, `fontStyle:'italic'`, `textDecoration:'underline'` based on marks
   - `'hardBreak'`: `React.createElement(Text, null, '\n')`
   - Prepends a `React.createElement(Text, { style: styles.h1 }, sectionName)` as the section title heading

2. `buildPdfBuffer(sections: Array<{ section_name: string; content: JSONContent | null }>): Promise<Buffer>` — iterates SECTION_NAMES order, calls `tiptapToPdfElements` for each, builds:
   ```
   React.createElement(Document, null,
     React.createElement(Page, { size: 'LETTER', style: styles.page }, ...allElements)
   )
   ```
   Then calls `renderToBuffer(doc)` and returns the result.

Imports:
- `import React from 'react'`
- `import { Document, Page, View, Text, StyleSheet, renderToBuffer } from '@react-pdf/renderer'`
- `import type { JSONContent } from '@tiptap/react'`
- `import { SECTION_NAMES } from '@/lib/editor/types'`

Key detail: Page size is `'LETTER'` (8.5" x 11") per UI-SPEC, NOT `'A4'` — US government submission standard.

```typescript
import { getUser, createClient } from '@/lib/supabase/server'
import { buildPdfBuffer } from '@/lib/export/tiptap-to-pdf'
import { stripComplianceMarks } from '@/lib/editor/compliance-gap-mark'

export const runtime = 'nodejs'  // CRITICAL — prevents edge runtime + react-reconciler conflict

export async function POST(
  _req: Request,
  { params }: { params: Promise<{ id: string }> }
) {
  const { id } = await params
  const user = await getUser()
  if (!user) return new Response('Unauthorized', { status: 401 })

  const supabase = await createClient()

  // Load proposal title for filename
  const { data: proposal } = await supabase
    .from('proposals')
    .select('title')
    .eq('id', id)
    .eq('user_id', user.id)
    .single()

  const { data: sections } = await supabase
    .from('proposal_sections')
    .select('section_name, content')
    .eq('proposal_id', id)
    .order('created_at', { ascending: true })

  if (!sections?.length) return new Response('No sections found', { status: 404 })

  // Strip compliance marks before conversion
  const cleanSections = sections.map(s => ({
    section_name: s.section_name,
    content: s.content ? stripComplianceMarks(s.content) : null,
  }))

  const buffer = await buildPdfBuffer(cleanSections)

  // Sanitize title for filename
  const title = proposal?.title ?? 'proposal'
  const safeFilename = title.replace(/[^a-zA-Z0-9-_ ]/g, '').replace(/\s+/g, '-').toLowerCase() || 'proposal'

  return new Response(buffer, {
    headers: {
      'Content-Type': 'application/pdf',
      'Content-Disposition': `attachment; filename="${safeFilename}.pdf"`,
    },
  })
}
```

Key details:
- `export const runtime = 'nodejs'` — CRITICAL for react-pdf
- Same auth + section load + compliance strip pattern as docx route
- `buildPdfBuffer` already returns a Buffer, no need to wrap
- Sanitized proposal title in filename
