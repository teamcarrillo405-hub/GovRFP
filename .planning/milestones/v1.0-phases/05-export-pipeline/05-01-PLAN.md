2. Update `next.config.ts` to add `serverExternalPackages`:

```typescript
import type { NextConfig } from "next";

const nextConfig: NextConfig = {
  serverExternalPackages: ['@react-pdf/renderer'],
};

export default nextConfig;
```

This is REQUIRED to prevent `TypeError: ba.Component is not a constructor` when `@react-pdf/renderer` runs in App Router route handlers. Without this, Next.js resolves React via the `react-server` condition which strips internals that `react-reconciler` needs.
