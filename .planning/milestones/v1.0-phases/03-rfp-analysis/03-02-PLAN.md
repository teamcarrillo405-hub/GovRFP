Export `computeWinScore(factors: WinFactors): number`:
```typescript
export function computeWinScore(factors: WinFactors): number {
  const raw =
    factors.scope_alignment.score * WIN_SCORE_WEIGHTS.scope_alignment +
    factors.certifications_match * WIN_SCORE_WEIGHTS.certifications_match +
    factors.set_aside_match * WIN_SCORE_WEIGHTS.set_aside_match +
    factors.past_performance_relevance.score * WIN_SCORE_WEIGHTS.past_performance_relevance +
    factors.competition_level.score * WIN_SCORE_WEIGHTS.competition_level
  return Math.round(Math.max(0, Math.min(100, raw)))
}
```

Export `computeCertificationsScore(contractorCerts: string[], rfpSetAsides: string[]): number`:
- If `rfpSetAsides.length === 0` return 50 (neutral — no preference stated)

- If any rfpSetAside matches any contractorCert after normalization, return 90
- Otherwise return 20

Export `computeSetAsideScore(contractorCerts: string[], primarySetAside: string | null): number`:
- If `primarySetAside` is null, return 50 (full and open — neutral)
- Normalize both primary and each cert
- If any cert matches primary, return 100
- Otherwise return 0
